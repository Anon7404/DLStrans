Help

Importent:

Only Textboxes and Decisions on Maps ,Items (inc. Armor/Weapons), Skills and Enemies are supported at the moment

How to Use with Maps:

Select a Map File with the File Selector on the Top (The Map Files are in You Game Dir under data, called Map001.json and so on).

Wait for the File to be Processed (may take some time, depending on your PC and the size of the File).

On the Left is a List with Events, there ID and there Pages, if an Event or a Page of it has no Text it won´t show up.

If you Select one of the Events you get the Main Text Table.

Here you can see the Original Content of the Textfield and one Field to Change it.

If you edit the New Field you NEED to click Somewhere else or press Enter to save the change.

Once the Changes are saved the Row turns Green.

How to Use with Items/Skills:

Just upload it, and change Stuff.

If you press the Download Button, obviously you Download the "New" Changed Version of the File, with the same Name as the Original one.

If you Upload a new File without saving the old one first, the Changes will be lost, also if you reload the Page.
